# Portfolio Animations

A stunning gooey animation effect built with WebGL and GSAP. Perfect for creating eye-catching portfolio landing pages.

## Features

- ✨ Interactive gooey/liquid blob effect
- 🎨 Customizable colors and animations
- 📱 Responsive design
- 🎮 GUI controls for real-time parameter adjustment
- 🔗 Smooth scroll-triggered animations
- ⚡ High performance WebGL rendering

## Technologies Used

- **WebGL** - For rendering the gooey effect
- **GSAP** - For scroll animations and timelines
- **lil-gui** - For interactive parameter controls
- **HTML5 Canvas** - For animation rendering

## How to Use

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Use the GUI panel to adjust animation parameters in real-time
4. Scroll to trigger animation effects

## Files

- `index.html` - Main HTML structure and shaders
- `style.css` - Styling and animations
- `script.js` - WebGL setup and JavaScript logic

## Browser Support

Requires a browser with WebGL support:
- Chrome/Edge ✅
- Firefox ✅
- Safari ✅
